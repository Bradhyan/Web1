Posibilidades de Visualización de JavaScript

En JavaScript se puede mostrar datos de diferentes formas:

1.innerHTML: Me sirve para escribir en un elemento de HTML(un elemento de HTML es una etiqueta)

2.document.write(): A diferencia del anterior, no se necesita especificar en que etiqueta se va a ejecutar, sino que se comporta como una

3.window.alert(): Me sirve para escribir en un cuadro de alerta

4.console.log(): Me sirve para mostrar un dato o un resultado en la consola de un navegador

innerHTML

Para acceder a un elemento de HTML(a una etiqueta) desde JavaScript se utiliza el document.getElementById, seguido del nombre del atributo id que tenga una etiqueta
Id como atributo define a que elemento voy a seleccionar y el innerHTML define que quiero escribir sobre ese elemento elegido
