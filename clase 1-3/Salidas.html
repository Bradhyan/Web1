Posibilidades de visualización de JavaScript


En javascript se puede mostrar datos diferentes formas:

1. innerHTML Me sirve para escribir en un elemento de HTML(En una etiqueta)
2. document.write():A diferencia del anterior no se necesita especificar en que etiqueta se va a ejecutar, sino que se comporta como una.
3. window.alert():Me sirve para escribir en un cuadro de alerta
4. console.log()Me sirve para mostrar un dato o un resultado en la consola de un navegador.

inner.HTML

Para acceder a un elemento de HTML desde JavaScript se utiliza el document.getElementById, seguido del nombre del atributo id que tenga una etiqueta
Id como atributo define que elemento voy a seleccionar y el innerHTMLdefine que quiero escribir sobre ese elemento elegido.
